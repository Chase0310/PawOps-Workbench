下一层: Runtime 进程拓扑

已确认 pnpm workspace。现在决定 AgentRuntime 在 Web 调试和桌面端里如何运行。

A

Embedded in Electron Main

桌面端直接在 Electron main process 中启动 runtime；Web 调试另起 dev server/runtime。

B

Local Runtime Daemon

runtime 是独立本地服务；Web 和 Desktop 都通过 HTTP/WebSocket 连接它。

C

Hybrid: Shared Runtime Package

runtime core 是纯 package；桌面可嵌入，开发可用本地 dev server 包一层 API。

推荐: C
packages/runtime-core: pure TypeScript runtime library
apps/desktop: Electron main imports runtime-core
apps/web: debug UI connects to dev API wrapper
packages/runtime-server: optional HTTP/WebSocket wrapper
