已确认: 产品技术栈选择 A

Electron + React + Vite + Node.js + TypeScript

Product Shape
Shared Workbench UI: React + Vite
Web Debug Mode: browser app connects to dev runtime
Desktop Mode: Electron packages UI and manages local runtime
Runtime Core: Node.js + TypeScript services
Adapters: coding CLI, MCP, storage, channels

Why this choice

  • AI-generated code quality is strongest in this stack.
  • Local process and stream handling are straightforward in Node.
  • One JS/TS stack keeps maintenance simpler than Python + JS + Rust.

Accepted tradeoff

  • Electron is heavier than Tauri.
  • We optimize for compatibility and development speed first.
