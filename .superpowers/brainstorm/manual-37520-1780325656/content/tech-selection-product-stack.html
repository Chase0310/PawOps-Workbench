技术选型: Web + Desktop 产品形态

目标是 Web 便于调试，桌面端便于分发和访问本地开发环境。

A

Electron + React + Node/TypeScript

最主流、AI 最熟悉、拉起本地 CLI 最顺。缺点是桌面包体和内存更大。

B

Tauri + React + Node Sidecar

桌面更轻，但会引入 Rust/Tauri 配置和 sidecar 打包复杂度。

C

Python Runtime + React Desktop Shell

Agent/AI 生态强，但 Web/desktop/CLI 分发会变成 Python + JS 双栈。

推荐: A
Shared UI: React + Vite
Web Debug App: browser connects to local/dev runtime
Desktop App: Electron main process hosts local runtime
Runtime Core: Node/TypeScript service modules
Adapters: coding CLI, MCP, channels, storage
