文档结构选项

先定义协作协议和文档骨架，再进入 AgentRuntime Core 的模块设计。

A

Module Bundle

每个模块一个目录，里面固定有 spec.md、tasks.md、checklist.md，必要时加 decisions.md。

B

Centralized Specs

所有 spec、tasks、checklist 分别放在 docs/specs、docs/tasks、docs/checklists，用文件名关联。

C

RFC + ADR

每个大改动先写 RFC，跨模块决策写 ADR，适合多人长期维护但早期更重。

推荐: Module Bundle + Lightweight ADR
