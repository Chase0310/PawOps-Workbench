技术选型: 第一层 Runtime 基线

先决定 TypeScript 运行时和工程基线，再讨论 UI、存储、事件、测试。

A

Node.js + pnpm

最稳妥的 TS 基线。生态、MCP、CLI 子进程、测试、打包、桌面/服务端扩展都成熟。

B

Bun First

开发体验快，内置工具多，但长期进程、兼容性和 MCP/CLI 边缘场景要额外验证。

C

Deno

权限模型和 TS 体验好，但 Node 生态、npm 包兼容、coding CLI 集成会更受约束。

推荐路径
Runtime: Node.js active LTS
Package manager: pnpm workspace
Language: TypeScript strict mode
Core packages: runtime-core, agent-profile, cli-adapter
Later: workbench UI, storage, channel adapters
